# Hotel Sentiment Analysis System

An AI-based sentiment analysis system that analyzes hotel customer reviews to identify customer satisfaction patterns and classify sentiments.

The project uses machine learning techniques to extract insights from customer feedback and provides interactive visualization through a Streamlit dashboard and Tableau reports.

## Features

- Sentiment classification of hotel reviews
- Machine learning model using XGBoost
- Customer feedback clustering
- Interactive Streamlit web interface
- Data visualization dashboard using Tableau
- Customer satisfaction analysis

## Technologies

- Python
- XGBoost
- Scikit-learn
- Pandas
- Streamlit
- Tableau
- Machine Learning
- Data Visualization

## Project Workflow

Data Collection → Data Cleaning → Feature Engineering → Model Training → Sentiment Prediction → Visualization

## Results

The system provides insights into customer opinions and helps businesses understand service quality and improve customer experience.
